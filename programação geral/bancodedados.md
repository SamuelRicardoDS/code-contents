# Banco de Dados

> Antigamente, com o surgimento dos computadores os dados eram armazenados fisicamente em fichas e afins.

> Com o advento da tecnologia, conceitos físicos foram utilizados na performance
> * Fichas passaram a se chamar registros
> * Pastas passaram a se chamar tabelas
> * gavetas ou armários passaram a se chamar arquivos
